Introductory Workshops

We know that this may be the first time you have participated in an informatics Olympiad. So, to help ease you into the competition, we have prepared some workshops so that you can learn the basics of competitive programming!

Code Complexity

date_range16 December, 2pm - 4pm

We want our programs to be efficient, but what does that mean? In this workshop, we will be exploring what makes a program efficient and how we can measure that efficiency.

Introduction to Data Structures

date_range18 December, 2pm - 4pm

Effective use of data structures complement good algorithms. But what are data structures and how can we use them? In this workshop, we will introduce common data structures, how they work under the hood and how they can be applied.
